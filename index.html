香港謎路 · 回憶機
Development Brief (MVP)

1. 核心概念 (Core Concept)

本項目為 **Mobile-first Web App**，結合實境探索與敘事解謎。玩家扮演「回憶解碼師」，在大角咀街道尋找隱藏的「回憶鎖匙」。

2. 遊戲流程 (User Journey)

  • Onboarding: 用戶輸入電話登入，確認 GPS 定位權限。
  • World Map: 使用 Geofencing 技術，玩家進入指定座標 30m 內觸發謎題。
  • Puzzle (D&D): 引入 Drag & Drop (拖放) 交互邏輯，模擬「修復回憶」的動作。
  • Memory Machine: 解鎖後的故事片段會存入「九宮格存檔」，供隨時回顧。

3. 技術棧 (Tech Stack)

Framework: Next.js (App Router)
Styling: Tailwind CSS
Interaction: Framer Motion / DnD Kit
Storage: LocalStorage (MVP)
